# 📊 Project Management Case Study Analyses

**ISM 555 — Project Management, Northern Arizona University (Spring 2025)**
**Team 3: Vamshi Aitharaju · Bhavani Durga Koukuntla · Navya Sri Katta · Jahnavi Chalapaka**

Three standalone case study analyses completed across the semester, each applying a different core project management discipline to a separate hypothetical business scenario.

`Business Case Development` `Earned Value Management` `Risk Management`

## The Cases

| # | Case Study | Focus | What It Demonstrates |
|---|---|---|---|
| 1 | **IT Infrastructure Modernization** | Business case development | Quantifying benefits, aligning a project to business objectives, cost-vs-benefit framing |
| 2 | **Budgeting for a Large-Scale Infrastructure Project** | Cost forecasting on a $600M–$950M scenario | Earned Value Management — CPI/SPI, budget compliance tracking, stable cash flow planning |
| 3 | **Risk Management for an ERP System Rollout** | Qualitative risk framework | Risk identification, probability-impact assessment, mitigation planning under an aggressive fixed deadline |

Each case is a different scenario and a different PM discipline, but the same underlying approach every time: structure the problem, quantify it, and plan explicitly around what could go wrong.
